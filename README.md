# Daily Routine

A minimalist habit tracking app with modern glassmorphism UI design.

## Features

- 📅 **Calendar-based tracking** - Check in daily habits with a monthly calendar view
- ✓ **Visual feedback** - Green checkmarks show completed days
- 📊 **Monthly stats** - Track completion rate for each habit
- 🎨 **Luxury glassmorphism UI** - Warm gradient background with frosted glass panels
- 🌐 **Cross-platform** - Built with Flutter for iOS, Android, Web, Linux, macOS, and Windows

## Tech Stack

- **Flutter** 3.10.1+
- **Noto Sans SC** typography (18px)
- **google_fonts** for font management
- **table_calendar** for calendar views

## Getting Started

```bash
# Install dependencies
flutter pub get

# Run the app
flutter run
```

## UI Design

Modern luxury glassmorphism style featuring:
- Warm sunset gradient background
- Backdrop blur effects with semi-transparent containers
- 30px rounded corners throughout
- Gold-to-orange gradient accents
- Floating 3D decorative elements
