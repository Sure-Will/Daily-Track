# 3D 装饰元素素材说明

此文件夹用于存放现代奢华玻璃拟态风格的 3D 装饰元素图片。

## 所需素材列表

为了实现完整的视觉效果，请准备以下透明背景 PNG 图片：

1. **marble_sphere.png** - 暖色调大理石纹理球体
   - 建议尺寸: 300x300px 或更高
   - 材质: 温暖的米色/橙色大理石纹理
   - 格式: PNG with transparency

2. **metal_ring.png** - 拉丝古铜/玫瑰金金属环
   - 建议尺寸: 400x400px 或更高
   - 材质: 玫瑰金或古铜金属质感
   - 格式: PNG with transparency

3. **stone_sphere.png** - 粗糙石质暗红球体
   - 建议尺寸: 250x250px 或更高
   - 材质: 暗红色粗糙石材
   - 格式: PNG with transparency

## 临时占位符

在获得真实素材前，代码会使用占位符路径。您可以：
- 使用简单的彩色圆形图片作为临时替代
- 从免费 3D 素材网站获取类似资源
- 使用 Blender 等 3D 软件自行渲染

## 推荐资源网站

- [Poly Haven](https://polyhaven.com/) - 免费 3D 素材
- [Sketchfab](https://sketchfab.com/) - 3D 模型资源
- [Freepik](https://www.freepik.com/) - 装饰性图形元素
